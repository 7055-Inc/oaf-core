# Diagonal Shift Template

**Version:** 1.0.0  
**Tier:** Professional

## Overview

Diagonal Shift rotates the entire layout 15° for a dynamic, unconventional presentation. Text counter-rotates for readability creating a visually striking experience.

## Key Features

- ↗️ 15° rotation of entire layout
- 🔄 Counter-rotated text (readable)
- 🎯 Diagonal or vertical scroll
- 📐 Dynamic visual tension
- ✨ Hover effects straighten cards

## Custom Fields

1. **Angle Direction** - right/left
2. **Scroll Direction** - diagonal/vertical
3. **Text Rotation** - counter/match

## Target Audience

- Dynamic artists
- Motion designers
- Contemporary portfolios
- Unconventional presentations

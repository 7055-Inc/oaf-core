# Polaroid Scatter Template

**Version:** 1.0.0  
**Tier:** Basic

## Overview

Polaroid Scatter displays products as scattered polaroid photos with tape holding them on a textured surface. Nostalgic, personal, and charming presentation.

## Key Features

- 📷 Polaroid photo aesthetic
- 📌 Tape/pin attachments
- 🪵 Textured backgrounds (cork/wood/fabric)
- ✍️ Handwriting-style captions
- 🎲 Random rotations
- 📐 Scatter density control

## Custom Fields

1. **Surface Texture** - cork/wood/fabric
2. **Show Tape** - yes/no
3. **Scatter Density** - tight/medium/loose
4. **Handwriting Style** - cursive/print/none

## Target Audience

- Personal portfolios
- Vintage aesthetics
- Casual presentations
- Memory-board vibes
- Nostalgic themes

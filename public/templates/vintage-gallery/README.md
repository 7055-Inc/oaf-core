# Vintage Gallery Template

**Version:** 1.0.0  
**Author:** Artist Sites  
**Tier Required:** Basic

## Overview

Vintage Gallery brings timeless, classic design to your artist site. With elegant serif typography, ornate frame options, and warm color tones, this template evokes traditional art galleries and museums, perfect for classical artists and fine art presentations.

## Target Audience

- Traditional artists
- Classical art galleries
- Fine art painters
- Artists preferring timeless, elegant presentation

## Design Philosophy

- **Timeless Elegance:** Classic design that never goes out of style
- **Frame Treatments:** Multiple ornate frame options for artwork
- **Vintage Tones:** Warm sepia, neutral cream, or cool gray color palettes
- **Museum Quality:** Spacious layouts inspired by gallery walls

## Custom Fields

### 1. Artwork Frame Style (`frame_style`)
- **Options:** Classic Frame, Ornate Frame, Simple Border, No Frame
- **Default:** Classic Frame
- **Adds visual frame treatment to artwork**

### 2. Color Tone (`color_tone`)
- **Options:** Warm Sepia, Neutral Cream, Cool Gray
- **Default:** Warm Sepia
- **Controls overall color palette**

### 3. Layout Density (`layout_density`)
- **Options:** Spacious (Museum), Comfortable, Compact (Salon)
- **Default:** Comfortable
- **Controls spacing between artworks**

## Features

✅ Vintage color tones and textures  
✅ Multiple frame styles  
✅ Elegant serif typography

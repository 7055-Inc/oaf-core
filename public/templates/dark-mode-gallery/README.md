# Dark Mode Gallery Template

**Version:** 1.0.0  
**Author:** Artist Sites  
**Tier Required:** Professional

## Overview

Dark Mode Gallery is a modern template optimized for displaying artwork against dark backgrounds. With automatic system preference detection and manual theme toggling, it provides the perfect viewing experience for digital art, photography, and contemporary work.

## Target Audience

- Digital artists
- Photographers
- Modern/contemporary artists
- Artists wanting sleek, modern presentation

## Custom Fields

### 1. Default Theme (`default_theme`)
- **Options:** Dark (Default), Light, Auto (System Preference)
- **Auto mode:** Detects visitor's system preference

### 2. Dark Mode Primary Color (`dark_primary`)
- **Type:** Color picker
- **Default:** #64ffda (teal)
- **Accent color for dark mode**

### 3. Contrast Level (`contrast_level`)
- **Options:** Normal, High, Maximum
- **Controls contrast intensity**

## Features

✅ Automatic dark/light theme detection  
✅ Manual theme toggle button  
✅ Theme persistence (localStorage)  
✅ System preference monitoring  
✅ Smooth theme transitions  
✅ Keyboard shortcut (Ctrl+Shift+D)  
✅ Mobile meta theme-color support  
✅ High contrast options  

## Modular Addon Support

**Updated:** This template now includes standardized addon integration points for Sprint 12 addons.

### Addon Sections Included:

1. **Category Products Showcase** (`.addon-category-showcase`)
   - 4-column product grid
   - Dark-themed cards with accent color highlights
   - Hover effects optimized for dark mode

2. **Instagram Feed** (`.addon-instagram-feed`)
   - 4-column grid layout
   - Teal accent overlay on hover
   - Alternating background sections

3. **Blog Cards** (`.addon-blog-cards`)
   - 3-column small card grid
   - Dark cards with rounded corners
   - Teal badges and meta information

### Integration

These sections are styled with the template's dark theme and accent colors. When Sprint 12 addons are built, they will automatically detect these classes and inject content.

**Template works without addons** - sections are simply empty until populated.

**Theme Compatibility:** All addon sections respect the light/dark theme toggle.

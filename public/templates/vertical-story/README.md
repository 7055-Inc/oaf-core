# Vertical Story Template

**Version:** 1.0.0  
**Tier:** Professional

## Overview

Vertical Story transforms your portfolio into a scrolling narrative experience. Timeline navigation, chapter-based layout, and scroll-triggered animations create an engaging story-driven presentation.

## Key Features

- 📖 Story-driven vertical layout
- ⏱️ Animated timeline progress
- 📑 Chapter-based sections
- ✨ Scroll reveal animations
- 🎭 Alternating or centered chapters
- 🎨 Elegant, editorial design

## Custom Fields

1. **Timeline Position** - left/center/right
2. **Chapter Style** - alternating/centered
3. **Scroll Reveal** - yes/no
4. **Timeline Color** - custom color

## Target Audience

- Narrative portfolios
- Editorial presentations
- Story-driven artists
- Long-form content
- Journey-based presentations

## Technical Features

- Intersection Observer API for scroll reveals
- Smooth timeline progress tracking
- Responsive chapter layouts
- Print-friendly formatting

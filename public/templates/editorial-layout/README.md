# Editorial Layout Template

**Version:** 1.0.0  
**Author:** Artist Sites  
**Tier Required:** Basic

## Overview

Editorial Layout brings magazine-quality design to your artist site. With large featured images, sophisticated typography, and story-driven content blocks, this template is perfect for artists who want to present their work with editorial polish and narrative depth.

## Target Audience

- Editorial photographers
- Artists with storytelling focus
- Fine art photographers
- Artists seeking sophisticated, magazine-style presentation

## Design Philosophy

- **Editorial Excellence:** Magazine-quality layout and typography
- **Story-Driven:** Content blocks designed for narratives
- **Large Format:** Generous image sizing for impact
- **Reading Experience:** Optimized typography and spacing

## Custom Fields

### 1. Featured Section Layout (`featured_layout`)
- **Options:** Full-Width Hero, Split Screen, Featured Grid
- **Default:** Full-Width Hero
- **Controls hero/featured section style**

### 2. Content Width (`content_width`)
- **Options:** Narrow, Standard, Wide
- **Default:** Standard
- **Controls maximum content width for readability**

### 3. Typography Style (`typography_style`)
- **Options:** Classic Editorial, Modern Sans-Serif, Luxury Serif
- **Default:** Classic Editorial
- **Changes font pairing and editorial feel**

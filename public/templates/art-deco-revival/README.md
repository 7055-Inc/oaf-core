# Art Deco Revival Template

**Version:** 1.0.0  
**Tier:** Professional

## Overview

Art Deco Revival brings 1920s geometric luxury to modern portfolios. Bold patterns, metallic accents, symmetrical layouts, and ornate borders create a Gatsby-esque experience.

## Key Features

- 📐 Geometric patterns & shapes
- 🏆 Gold/emerald/navy palettes
- ◆ Ornamental borders & decorations
- 🔶 Clipped polygon shapes
- ✨ Symmetrical layouts
- 🎭 Vintage luxury meets modern

## Custom Fields

1. **Deco Palette** - gold-black/emerald-gold/navy-gold
2. **Geometric Style** - subtle/moderate/bold
3. **Show Ornaments** - yes/no

## Target Audience

- Luxury brands
- Vintage aesthetics
- High-end portfolios
- Fashion collections
- Art collectors

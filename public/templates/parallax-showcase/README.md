# Parallax Showcase Template

**Version:** 1.0.0  
**Author:** Artist Sites  
**Tier Required:** Professional

## Overview

Parallax Showcase brings dynamic, modern scrolling effects to your artist site. With smooth parallax backgrounds, fade-in animations, and depth-based scrolling, this template creates an engaging, immersive experience perfect for contemporary artists.

## Target Audience

- Modern/contemporary artists
- Digital artists
- Artists wanting dynamic, engaging presentation
- Creative professionals seeking modern aesthetics

## Custom Fields

### 1. Parallax Effect Intensity (`parallax_intensity`)
- **Options:** Subtle, Medium, Dramatic
- **Controls strength of parallax movement**

### 2. Scroll Animations (`scroll_animation`)
- **Options:** Enable, Disable
- **Fade-in animations as elements enter viewport**

### 3. Hero Parallax Background (`hero_parallax_image`)
- **Type:** Image URL
- **Background image for parallax hero**
- **Recommended: 1920x1080px**

## Features

✅ Parallax scrolling backgrounds  
✅ Viewport-triggered fade animations  
✅ Auto-hiding header on scroll  
✅ Staggered card animations  
✅ Performance-optimized (requestAnimationFrame)  
✅ Respects prefers-reduced-motion  
✅ Mobile-optimized (disables parallax)  
✅ GPU-accelerated transforms  

## Modular Addon Support

**Updated:** This template now includes standardized addon integration points for Sprint 12 addons.

### Addon Sections Included:

1. **Instagram Feed** (`.addon-instagram-feed`)
   - 4-column grid with gradient hover overlays
   - Parallax-themed styling
   - Positioned between main content and footer

2. **Blog Cards** (`.addon-blog-cards`)
   - 3-column small card grid
   - Gradient badges and hover effects
   - Light background section

3. **Featured Quote** (`.addon-quote-section`)
   - Full-width gradient background
   - Italic quote styling
   - Centered layout

### Integration

These sections are styled and ready. When Sprint 12 addons are built, they will automatically detect these classes and inject content, which will be styled by this template's CSS.

**Template works without addons** - sections are simply empty until populated.

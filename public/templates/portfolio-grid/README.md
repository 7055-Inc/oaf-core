# Portfolio Grid Template

**Version:** 1.0.0  
**Author:** Artist Sites  
**Tier Required:** Basic

## Overview

Portfolio Grid is a masonry-style image grid template designed for photographers and visual artists who want their work to be the star. With multiple grid layout options and customizable captions, this template creates a clean, gallery-focused presentation that lets your artwork shine.

## Target Audience

- Photographers
- Visual artists with strong image portfolios
- Illustrators
- Artists preferring minimal text, maximum artwork focus

## Design Philosophy

- **Image-First:** Artwork takes center stage
- **Clean & Simple:** Minimal distractions from the content
- **Flexible Layouts:** Multiple grid styles to suit different artwork
- **Lightbox-Ready:** Hover effects prepare for future lightbox integration

## Custom Fields

### 1. Grid Style (`grid_style`)
- **Options:** Masonry, Uniform Grid, Justified Rows
- **Default:** Masonry
- **Masonry:** Pinterest-style varied heights
- **Uniform:** Equal-sized grid cells
- **Justified:** Flexible justified rows

### 2. Grid Gap Size (`grid_gap`)
- **Options:** Compact, Medium, Spacious
- **Default:** Medium
- **Controls spacing between grid items**

### 3. Show Image Captions (`show_captions`)
- **Options:** Always Show, Show on Hover, Never Show
- **Default:** Show on Hover
- **Controls artwork title/price visibility**

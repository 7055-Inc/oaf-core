# Split Screen Template

**Version:** 1.0.0  
**Author:** Artist Sites  
**Tier Required:** Professional

## Overview

Split Screen creates an interactive dual-panel layout with a draggable divider. Perfect for before/after comparisons, portfolio presentations, or dual-content showcases, this template offers a unique and engaging way to present your work.

## Target Audience

- Portfolio artists
- Before/after showcases
- Dual-concept presentations
- Artists wanting unique, interactive layouts

## Custom Fields

### 1. Default Split Position (`default_split`)
- **Options:** 30/70, 50/50, 70/30
- **Initial divider position**

### 2. Split Orientation (`split_orientation`)
- **Options:** Vertical (Left/Right), Horizontal (Top/Bottom)
- **Direction of the split**

### 3. Allow User Resize (`allow_resize`)
- **Options:** Yes (Draggable), No (Fixed)
- **Let visitors drag the divider**

## Features

✅ Interactive draggable divider  
✅ Vertical or horizontal split  
✅ Touch-friendly (mobile support)  
✅ Keyboard shortcuts (Ctrl+Arrows)  
✅ Smooth dragging animations  
✅ Clamped range (20%-80%)  
✅ Auto-stacks on mobile  
✅ Optional fixed positioning  

## Keyboard Shortcuts

- **Ctrl/Cmd + Left/Right Arrow:** Adjust split position
- **Ctrl/Cmd + 0:** Reset to 50/50

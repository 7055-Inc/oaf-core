# Brutalist Studio Template

**Version:** 1.0.0  
**Tier:** Basic

## Overview

Brutalist Studio embraces raw, unpolished web design. Giant typography, thick borders, minimal color, and a terminal-inspired aesthetic create a bold, anti-corporate statement.

## Key Features

- 📏 GIANT typography (10rem+ headings)
- ⬛ Thick black borders everywhere
- 🎨 Single accent color only
- 🖥️ Terminal-inspired footer
- 🔲 Grid without gaps
- ⚫ Black/white + one accent

## Custom Fields

1. **Accent Color** - Single highlight color
2. **Show Borders** - yes/no
3. **Typography Weight** - bold/black
4. **Link Style** - thick/double/none underline

## Target Audience

- Experimental artists
- Underground/punk aesthetic
- Anti-corporate stance
- Raw/unpolished vibes
- Minimalist extremists

# Broken Grid Template

**Version:** 1.0.0  
**Tier:** Professional

## Overview

Broken Grid intentionally breaks traditional grid layouts with random rotations, overlapping elements, and collage-style aesthetics. Perfect for experimental artists who want to challenge conventions.

## Key Features

- 🎲 Random rotations (-8° to +8°)
- 📐 Overlapping cards (optional)
- 📌 Masking tape visual effect
- 🎨 Collage/mood board aesthetic
- ⚙️ Configurable chaos levels

## Custom Fields

1. **Chaos Level** - subtle/medium/wild
2. **Rotation Range** - ±3°/5°/8°
3. **Overlap Style** - yes/no
4. **Show Tape Effect** - yes/no

## Target Audience

- Experimental artists
- Contemporary art
- Collage artists
- Anti-establishment vibes
- Zine aesthetics

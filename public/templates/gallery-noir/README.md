# Gallery Noir Template

**Version:** 1.0.0  
**Tier:** Professional

## Overview

Gallery Noir creates a luxurious, museum-quality dark gallery experience. Spotlight effects, metallic accents, and refined typography present artwork like pieces in a high-end private collection.

## Key Features

- 🖤 Dark, luxurious aesthetic
- ✨ Spotlight/vignette effects
- 🏆 4 metallic accent options (gold/rose gold/silver/platinum)
- 🖼️ Ornate frame options
- 📐 Museum-quality spacing
- 🎭 Smooth reveal animations
- 🎨 Gradient metallic text effects

## Custom Fields

1. **Accent Metal** - gold/rose-gold/silver/platinum
2. **Spotlight Intensity** - subtle/medium/dramatic
3. **Card Frame Style** - none/thin/ornate
4. **Typography Style** - serif/modern

## Target Audience

- High-end artists
- Fine art photographers
- Luxury brands
- Gallery-quality portfolios
- Premium collections

## Technical Features

- Radial gradient backgrounds
- Backdrop blur effects
- CSS gradient text (metallic)
- Smooth cubic-bezier transitions
- Intersection Observer animations
